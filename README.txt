Точка входа - run_bot.py

Бот реагирует на добавление / удаление из чата.
После добавления в чат данный чат добавится в БД и появится приветственное сообщение с информацией,
как запустить приложение.
При удалении бота из чата удаляется соответствующая запись из БД
(в БД хранится информация только о тех чатах, куда бот был добавлен)

После запуска бота командой /start можно приступать к регистрации игроков.
Участвовать в игре (голосовать за фото) могут только зарегистрированные игроки.
В БД хранится информация обо всех игроках – как зарегистрированных на текущую игру в конкретном чате, так и обо всех,
кто когда-либо регистрировался (общий перечень user’ов). При регистрации участника бот направит персональное
всплывающее уведомление игроку. Участвовать в игре может только четное количество участников –
бот проинформирует участников чата, если количество зарегистрированных на игру игроков будет нечетным.

После успешного завершения процесса регистрации можно запустить текущий игровой раунд.
Процесс проведения игровых раундов:
- Из общего пула зарегистрированных игроков случайным образом выбирается по два игрока
- Участники игры голосуют за понравившийся аватар игрока
- По результатам голосования определяется победитель, который остается в игре на следующий раунд.
  Проигравший в голосовании игрок выбывает из участия
- В случае ничьи оба игрока продолжают свое участие
- Игра продолжается до тех пор, пока среди игроков не останется один участник – победитель
По результатам раунда / игры бот проинформирует участников чата.
Всем принимавшим в игре участникам бот начислит участие в игре, победителю – победу.

Примечания по игровому процессу:
- Проголосовать может только зарегистрированный на текущую игру (в конкретном чате) игрок
- Дважды проголосовать зарегистрированный игрок не сможет
* бот проинформирует игрока как об успешном голосовании, так и о невозможности проголосовать по указанным выше причинам

На каждом этапе можно войти в данный этап игры командой /start – состояние отслеживается в БД.
Аналогично, на разных этапах можно и выйти из игры (вернуться на начальный этап).
